CS180: Intro to Computer Vision and Computational Photography
Fall 2025

Project Overview

This project implements an automatic colorization algorithm for the digitized Prokudin-Gorskii glass plate images.

Approach

My implementation uses a multi-scale pyramid approach for efficient and accurate alignment:

  • Channel Splitting: Divides the input image into three equal parts (B, G, R channels)
  • Image Pyramid: Creates multi-scale representations for coarse-to-fine alignment
  • Gradient-based Matching: Uses gradient magnitudes instead of raw pixel values for robust alignment
  • Normalized Cross-Correlation (NCC): Measures alignment quality between channels

Algorithm Details

The algorithm works by:

  1. Building image pyramids by downsampling by factors of 2
  2. Starting alignment at the coarsest level with a large search radius
  3. Refining alignment at each finer level with smaller search windows
  4. Using gradient magnitudes to compute NCC scores for better feature matching
  5. Cropping borders to avoid edge artifacts during alignment
  6. Stacking aligned channels into final RGB image
  • For single-scale implementation, set the pyramid levels to 1.
  • The offsets are integer, pixel-wise translations wrt. the Blue channel.
